# ClearVue Ltd. – NoSQL Group Project

This repository contains the group project for ClearVue Ltd., focusing on building a **NoSQL database solution**, **streaming pipelines**, and **business intelligence (BI) dashboards**, along with a **final report** documenting the system.

---

## 📌 Project Overview
ClearVue Ltd. requires a data solution to manage and analyze sales and operational data efficiently.  
Our group is developing a **prototype system** using NoSQL technologies (e.g., MongoDB), data streaming pipelines, and BI visualization dashboards.  
A final report will document the **needs, solution, implementation, and reflections**.
